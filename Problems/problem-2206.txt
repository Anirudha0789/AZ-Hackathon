
Jon is the fastest man alive. He and you are in a competition.
Both of you are given three integers AA, BB, and NN.
Travel from 11 to NN.

If the number is a multiple of AA, print “Beautiful”.
If the number is a multiple of BB, print “Lie”.
If the number is a multiple of both AA and BB, print “BeautifulLie”.
Else print the current number itself.

As you know, you have to be fast, so print all the expected results as fast as you can to beat Jon.
Input Format

Input, three integers A,B,NA,B,N.

Output Format

Print "Beautiful", “Lie”, “BeautifulLie” or THE NUMBER ITSELF as accordance each in new line.

Constraints

1≤N≤1041≤N≤104
1≤A<B≤N1≤A<B≤N
