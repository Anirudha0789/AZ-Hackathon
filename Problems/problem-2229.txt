
Sherin likes to play with numbers. Sherin decides to make a pyramid using numbers.Help Sherin to make the pyramid.
Input:
First line contains a single integer T,Number of test cases.
Every test case contains an integer R,Number of rows.
Output:
For each test case ,print the pyramid.
Constraints
1<=T<=10
1<=R<=10^4
Sample Input:
1
3
Sample Output:
1
2 3 2
3 4 5 4 3
