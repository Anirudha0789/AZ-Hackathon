
You are given a string of characters and you have to find the Sum of the indexes of each character (according to lexicographical order). check if the sum of all indexes is even or odd.
If the sum of all index values is even then print "YES" (without quotes). else print "NO"(without quotes).
NOTE:NOTE: The string contains only lowercase characters.
Input:

First line will contain TT, number of testcases. Then the testcases follow. 
Each testcase contains of a single line of string SS.

Output:
For each testcase, output in a single line answer i.e "YES" or "NO" (without quotes).
Constraints

1≤T≤10001≤T≤1000
1≤stringsize≤10001≤stringsize≤1000

Sample Input:
1
abhishek

Sample Output:
NO 

Explaniation:
Testcase1:Testcase1: The sum of all index values for the given string is 63 (1 + 2 + 8 + 9 + 19 + 8 + 5 + 11) so the answer is NONO.
