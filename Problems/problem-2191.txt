
You are given the marks of student A and B out of 500.
you have to calculate their percentage. if the percentage of student A is greater then B the print "A" without quotes else print "B".
Input:

First line will contain TT, number of testcases. Then the testcases follow. 
For each testcase the first line contains an integer A&BA&B, denoting the total marks of student 
A and student B.

Output:
For each testcase, output in a single line answer containing the name of student who scored more percentage.
Constraints

1≤T≤10001≤T≤1000
1≤A,B≤1061≤A,B≤106

Sample Input:
3
400 250
320 499
200 400
Sample Output:
A
B
B
